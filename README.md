## 🚀 使用建议

### 面试准备路线

1. **基础阶段**：阅读 01-Foundation，建立大模型整体认知
2. **进阶阶段**：深入学习 02-05，理解训练、评估、应用全流程
3. **算法巩固**：按分类刷 LeetCode 题目，重点掌握高频考点
4. **实战演练**：准备 Resume，结合项目经验练习面试回答

### 刷题建议

- 按分类系统学习，先易后难
- 重点理解解题思路，而非死记硬背
- 定期复习，巩固记忆
