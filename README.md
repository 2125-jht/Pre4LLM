### 🤖 MiniMind - 从零开始的大模型

[MiniMind](https://github.com/jingyaogong/minimind) 是一个开源的轻量级大语言模型项目，**大道至简**，旨在以极低成本（3块钱+2小时）从零开始训练一个参数仅25.8M的超小语言模型。

**项目特点：**
- 🪶 **极致轻量**：最小版本体积仅为 GPT-3 的 1/7000
- 💰 **成本极低**：普通个人GPU即可快速训练
- 🔧 **完整流程**：包含预训练、SFT、LoRA、DPO、RLAIF(PPO/GRPO)、模型蒸馏等全阶段代码
- 🖼️ **多模态支持**：同时支持 VLM 视觉语言模型（MiniMind-V）
- 🏗️ **原生实现**：所有核心算法使用 PyTorch 原生重构，不依赖第三方抽象接口

**适合人群：**
- 想深入理解大模型原理的初学者
- 需要快速上手LLM训练流程的开发者
- 准备大模型算法岗面试的求职者

> 本项目中的 `minimind/` 目录为该开源项目的本地副本，用于学习和实践。

## 🚀 使用建议

### 面试准备路线

1. **基础阶段**：阅读八股系列，建立大模型整体认知
2. **项目实践**：学习 minimind 项目，理解大模型训练全流程
3. **算法巩固**：刷 leetcode 题目，重点掌握高频考点
4. **实战演练**：结合项目经验练习面试回答

### 学习建议

- 八股文结合 minimind 源码理解，理论与实践结合
- 按分类系统刷题，先易后难
- 定期复习，巩固记忆
